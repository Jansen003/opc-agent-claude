# OPC 知识库检索

## 搜索路径

知识库根目录：$OPC_KNOWLEDGE_PATH（默认 /Users/souljian/code/opc/opc-knowledge）

| 目录 | 搜索内容 | 场景 |
|------|----------|------|
| 01-ADR/ | 架构决策记录 | 技术选型、架构变更 |
| 02-PRD/ | 产品需求文档 | 需求确认、功能范围 |
| 03-Design/ | 技术设计方案 | 实现方案、接口设计 |
| 04-Solutions/ | 解决过的非显然问题 | 遇到类似问题时 |
| 05-Competitor/ | 竞品分析 | 市场定位、差异化 |
| 08-Lessons/ | 踩坑记录 | 避免重复犯错 |
| 09-Conversations/ | 会议纪要 | 历史讨论、决策依据 |
| 10-Tasks/ | 任务看板 | 当前进度、待办 |

## 搜索规则

1. 讨论任何议题前，先搜索知识库是否有相关记录
2. 搜索关键词：从议题中提取核心名词
3. 找到相关记录后，出示摘要：「关于此议题，知识库中有以下历史记录：[摘要]」
4. 创始人决定：基于历史继续讨论 / 忽略历史重新讨论

## 搜索方式

优先用 mcp-obsidian 工具搜索，MCP 不可用时直接 grep 文件。
